# Project Management Approach

This project follows a hybrid approach combining elements of Scrum and Kanban.

Development is iterative, with small increments of functionality added step by step. Tasks are managed flexibly, allowing adjustments based on progress and feedback.

## Responsibilities

Human (Developer):

* Define architecture
* Review AI-generated code
* Make final design decisions

AI Assistant:

* Generate boilerplate code
* Suggest improvements
* Help with testing
* Assist in debugging

This division allows efficient development while maintaining control over code quality and system design.
